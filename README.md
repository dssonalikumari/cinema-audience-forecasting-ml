# Cinema Audience Forecasting using Machine Learning

## Overview
This project predicts daily cinema audience count using historical booking trends and theatre metadata.

## Features Used
- Lag Features
- Rolling Mean Features
- Weekday/Weekend Features
- Lead Time Features
- Booking Aggregations

## Models Used
- Ridge Regression
- LightGBM
- XGBoost

## Evaluation Metrics
- RMSE
- MAE

## Tools & Libraries
- Python
- Pandas
- NumPy
- Scikit-learn
- LightGBM
- XGBoost
- Matplotlib
- Seaborn

## Project Outcome
XGBoost achieved the best forecasting performance on the validation set.
